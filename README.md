# SuitCut

Modern minimalist PWA for bespoke suit order management built with React, TypeScript, Vite, and Material UI.

## Features
- Mobile-first monochromatic interface with sticky header and primary bottom navigation for quick access.
- Sales Order Form with stylus/handwriting canvas that saves as an image attachment.
- Support for multiple sales agents and tailor shops with status-driven lifecycle (Pending Acceptance, Accepted, In Progress, Completed, Sample Image Sent, Shipped, Paid).
- Dashboard health cards and order list with quick status advancement.
- PWA-ready: manifest, custom service worker, mobile-friendly install experience.

## Getting started
```
npm install
npm run dev
```

If the registry is restricted in your environment, install dependencies from an accessible mirror before running the commands above.
